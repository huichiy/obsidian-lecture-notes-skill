# obsidian-lecture-notes-skill
Generate Obsidian-ready revision notes from university lecture materials such as PDFs, PPTs, slides, screenshots, or pasted text. Use for lecture content, course materials, chapter notes, or revision-note requests. Output one Markdown file with YAML frontmatter, Mermaid diagrams, Obsidian wikilinks, flashcards, cheat sheet, and self-test.
